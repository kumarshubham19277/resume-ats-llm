# AI Resume ATS Checker

## 🚀 Project Overview
This project analyzes resumes against job descriptions and provides an ATS score along with AI-based suggestions.

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn (TF-IDF)
- Groq API

## 📊 Features
- Resume parsing (PDF/DOCX)
- ATS score calculation
- AI suggestions
- Interactive UI

## ▶️ Run Locally
```bash
streamlit run app.py
